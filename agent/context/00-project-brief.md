# Project Brief – TTC MARCONI Homepage

## Cíl projektu

Vytvořit prezentační homepage typu "scroll-as-story" pro TTC MARCONI, která působí technicky, důvěryhodně a přesně, bez marketingového balastu.

## Charakter produktu

- B2B enterprise web
- obsahově vedená homepage
- vysoký důraz na SEO, čitelnost a výkon
- cílový dojem: precizní systémová integrace, stabilita, kompetence

## Klíčové značkové principy

- White-first UI
- Tyrkys (`#01B5E5`) jen jako aktivní signál
- Tmavé sekce pouze pro důraz
- Žádná dekorativnost, žádné výrazné gradienty
- Interaktivita = reakce systému, ne vizuální show

## Primární obsahový záměr homepage

Homepage má v logické posloupnosti vysvětlit:

1. Kdo jsme a co děláme
2. Jak pracujeme (infrastruktura / diagram)
3. Kde působíme (sektory)
4. Proč nám věřit (zkušenost, certifikace, synergie TTC)
5. Jak nás kontaktovat

## SEO a performance priority

- Sémantická struktura H1-H3
- Obsahová úplnost bez závislosti na JS
- Minimal JS baseline (Astro)
- Interaktivita jen ve vybraných sekcích (islands)
- Optimalizace obrázků (AVIF/WebP, srcset)

## Ne-cíle (z aktuálního zadání)

- Není cílem flashy marketingový microsite styl
- Není cílem barevně experimentovat
- Není cílem stavět JS-heavy SPA

## Otevřené body k potvrzení (do budoucna)

- Reálné reference/loga/obrázky a práva k použití
- Finální copy (CZ/EN varianty)
- Přesná podoba diagramu infrastruktury (vizuál + datový model)
