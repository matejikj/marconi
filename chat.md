# 1️⃣ Tech stack & koncepty

### ❓ Já:

Máš stack Astro + Tailwind + GSAP/Lenis (islands). Chceš scroll-as-story homepage pro TTC MARCONI. Navrhni 3 koncepty včetně:

* 10 bodů vzhledu
* tokenů
* komponent
* motion
* responsiveness
* promptu pro vizualizaci

### ✅ Chat odpověděl:

Navrhl jsem 3 směry:

1. **Signal Flow** – tmavý high-tech, zářící síťové motivy
2. **Editorial Authority** – světlý, corporate, důvěra
3. **Immersive Infrastructure** – filmový, dramatický, silně interaktivní

Doporučil jsem hybrid 1 + 2 jako ideální kompromis.
